# Sükût Shared

Mobil uygulama ve admin panelin ortak kullandığı TypeScript paketi.

## İçerik

- Ortak domain tipleri
- Zod içerik şemaları
- Firestore koleksiyon sabitleri
- Türkçe i18n başlangıç dosyası
- Türkçe locale uyumlu arama/slug helperları
- Mojibake validator
- Offline seed içerik
- Desteklenen şehir metadata'sı ve prayer provider yardımcıları için ortak sabitler
- Streak ve rozet kazanımı için merkezi progress engine
- Gizlilik ve dini içerik disclaimer sabitleri
