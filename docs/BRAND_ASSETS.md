# Marka ve Asset Hazırlığı

App adı `Sükût`, bundle/package id `com.sukut.app` olarak güncellendi.

## Gerekli Assetler

- App icon: `apps/mobile/assets/icon.png` - kullanıcı tarafından verilen yazılı Sükût ikonundan 1024x1024 üretildi.
- Adaptive Android icon: `apps/mobile/assets/adaptive-icon.png` - kullanıcı tarafından verilen yazısız hilal/dağ ikonundan 1024x1024 üretildi.
- Splash screen: `apps/mobile/assets/splash.png` - kullanıcı tarafından verilen dikey splash görseli kullanıldı.
- Notification icon: `apps/mobile/assets/notification-icon.png` - yazısız ikon 96x96 olarak hazırlandı.
- Netlify icon: `netlify/icon.png` - ana ikonla eşlendi.
- Orijinal kaynaklar: `apps/mobile/assets/brand/`
- Store ekran görüntüleri
- Admin panel favicon

## Tasarım Notu

Klasik yeşil/aşırı altın dini uygulama görünümünden kaçınılır. Palet ve tasarım tokenları mobilde `src/theme/tokens.ts` içinde merkezi tutulur.

Aktif asset dili: gece mavisi zemin, yumuşak krem hilal, katmanlı dağ silüeti, göl yansıması ve serif `Sükût` yazımı. Store öncesi farklı cihaz preview kontrolü önerilir.
